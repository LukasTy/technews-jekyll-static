<p>Atvirąjį kodą propaguojanti bendruomenė Mozilla išleido trečiąją, kodiniu pavadinimu Fennec žinomos interneto naršyklės Alpha versiją, kuri yra skirta mobiliesiems įrenginiams. Tiesa, po oficialaus starto, pastaroji turėtų būti žinoma tiesiog kaip FireFox Mobile.</p>
<p>Naujoji versija galima Windows Mobile 6 operacine sistema grįstiems išmaniesiems telefonams ir turi daugybę patobulintų bei naujai įdiegtų savybių. Mozilla žada greitesnį naršyklės pasileidimą, kiek padidintus puslapio „piešimo“ bei slinkimo greičius, taip pat pilno ekrano palaikymą ir visišką suderinamumą su lietimui jautrų ekraną turinčiais įrenginiais.</p>
<p>Visi Windows Mobile išmaniuosius telefonus turintys vartotojai išbandyti Mozilla produktą gali užsukę <a class="ns" href=" http://download.mozilla.org/?product=fennec-1.0a3&os=winmo&#9001;=en-US">čia</a>, o Windows ir Mac vartotojai – <a class="ns" href=" http://www.mozilla.org/projects/fennec/1.0a3-wm/releasenotes/#start">čia</a>.<br /></p>
