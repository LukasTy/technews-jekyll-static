<p>Jau veikiausiai kitais metais pasirodysiančioje naujojoje korporacijos Microsoft operacinėje sistemoje, šiuo metu žinomoje Windows 8 vardu, bus pasiūlyta nauja ir integruota PDF failų skaitymo programa, kuri sudarys konkurenciją Adobe produktui.</p>
<p>Kaip žinia, šiuo metu norėdamas redaguoti ar peržiūrėti PDF failus, dažnas vartotojas atsisiunčia Adobe Reader programinę įrangą, nes Windows 7 neturi integruoto tokių failų skaitytuvo. Kaip pranešama, Microsoft Modern Reader turėtų viską pakeisti.</p>
<p>Manoma, jog Microsoft Modern Reader pasieks ir mobiliąją Windows Phone 7 operacinę sistemą, o galbūt iškart gandais apipintą artėjančią Windows Phone 8. Tai reiškia, jog peržiūrėti ir redaguoti PDF failus vartotojams nebereikės papildomų taikomųjų programų, integruotą pasiūlys Microsoft.<br /></p>
